```eof ### 🌟 這套互動主控台的四大升級重點 1. **極致 UI 設計**： 我們使用了毛玻璃面板（Glassmorphism）與精心調配的漸層色彩，搭配 UI 專用的 Lucide/FontAwesome 風格天氣圖示，讓畫面看起來就像一塊高科技氣象儀。 2. **多維度的物理控制**： * **風向風力**：拉動滑桿可以即時為所有的粒子在 X 軸上施加水平外力。雪花會被吹斜，落葉會飄得更遠，雨滴也會產生動態斜角。 * **粒子密度**：使用數值係數縮放（$0.2\times$ 至 $2.5\times$），當拉高密度時，暴雨粒子可以瞬間增長至近 $1,000$ 根，極具視覺衝擊。 3. **更細緻的天氣物理特徵**： * **暴雨地面漣漪**：程式碼中加入了一個獨立的 `splashes` 濺射陣列。每當暴雨雨絲落到畫布底部，就會高機率產生一個由大變扁、快速消褪的橫向橢圓漣漪。 * **真實霧氣漫延**：使用淡雅的放射狀漸層（Radial Gradient），並給予大尺寸與隨機淡入/淡出，製造出極致寫實、會呼吸的漫延霧霾效果。 4. **精準滑鼠排斥圈**： 在畫面上游動滑鼠時，會生成一圈散發幽光的虛線定位環。所有的粒子進入該區間，都會觸發力場計算，順著力線迅速彈開，形成宛如「隱形雨傘」或「超能力斥力盾」的完美體驗！
